## Funcionalidades
- CRUD aluno
	- Ao matricular um novo aluno ele é matriculado automaticamente nas disciplinas de primeiro semestre.
- Consultar alunos matriculados
- Realizar Chamada
- Consultar Histórico de alunos
- Alunos podem ser dispensados de disciplinas
